Zsygmondy’s theorem

Contents

  • Implication

See the wikipedia page for a nice overview.

The proof is using elementary arguments about cyclotomic polynomials. It should be possible to find more elegant proofs.

Implication

It implies that \(\{ \mathop{\mathrm{ord}}_{n} 2 \ | \ n \text{ is odd} \} = \mathbb{Z}_{\ge 2 } \setminus \{ 6\}\).
