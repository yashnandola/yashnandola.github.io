(Hasse, 1966, [1])

Define \(\pi_{2}(x)\) as \[\pi_{2}(x) = \#\{ p \ | \ 2 < p \le x\text{ is prime and } p|(2^{m}+1)\text{ for some }m \in \mathbb{Z}_{\ge 0}\}.\] This is the same set as \(\{ p \ | \ 2 < p \le x\text{ is prime and } \text{$\mathop{\mathrm{ord}}_p$}2\text{ is even} \}\).

Then, we have that \[\begin{align} \pi_{2}(x) = \frac{17}{24} \frac{x}{\log x} + o\left(\frac{x}{\log x}\right). \end{align}\]

Computational aspects

Here is a sage program to list such primes upto \(N\).

N=10000

for p in prime_range(3,N):
    ord_mp = multiplicative_order(mod(p,2))
    if ord_mp%2==0:
        print p

For sphere packing

This code generates an array containing division algebra packings used in the division algebra preprint

N=300

more=2.7

pack_ratio=3.0
log_d_prod=log(8)

g_packing=[]

for p in prime_range(3,N*more):
    ord_mp = multiplicative_order(mod(2,p))
    if ord_mp%2==1:
        pack_ratio*=1.0/(1-1.0/p)
        log_d_prod+=log(p-1)
        g_packing+=[[log_d_prod.n(),pack_ratio.n()]]

Lucas numbers

The result generalizes to Lucas numbers or primes dividint \(a^n + 1\) for some \(n\) and a fixed \(a\). See [2].

The set of primes that divide \(p^n + 1\) for some prime \(p\) has density \(2/3\) times the natural density of primes whenever \(p \ge 3\).
