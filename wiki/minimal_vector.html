Minimal Vector of a Lattice

Contents

  • Relation to SVP

The minimal vector length of a lattice is a continuous function on the space of lattices. Since space of lattices is \(GL_n(\mathbb{R})/GL_n(\mathbb{Z})\) with the continuous topology, the following lemma is sufficient to show this.

The following is a continuous function on \(GL_n(\mathbb{R})\). \[\begin{align} g \mapsto \min_{x \in \mathbb{Z}^{n} \setminus \{ 0\}} \| gx\|. \end{align}\]

The function is basically the \(\min\) over a countable set of functions. However for any point \(g_0 \in GL_n(\mathbb{R})\) the function is equal to a \(\min\) over finitely many functions in an open neighbourhood of \(g_0\).

Take \(\Omega\) to be any relatively compact set around \(g_0\). Fix a vector \(x_0 \in \mathbb{Z}^{n} \setminus \{ 0\}\). For a \(g \in \Omega\), the output of the function above will be \(\|g x_g\|\) where \(x_g \in \mathbb{Z}^{n} \setminus \{ 0\}\) is a vector which satisfies \[\begin{align} \|g x_g\| \le \| g y\|, \ \forall y \in \mathbb{Z}^{n} \setminus \{ 0\}. \end{align}\] In particular, \(\|g x_g\| \le \|g x_0\|\) and therefore for some constant \(R_{\Omega} > 0\) we have \[\begin{align} \|x_{g}\| = \|g^{-1} g x_{g}\| \le \|g^{-1}\| \|g x_{g}\| \le \|g^{-1}\| \|g x_{0}\| \le R_{\Omega}. \end{align}\] Here, \(\|g^{-1}\|\) is the operator norm on \(g^{-1}\). Hence, for \(g \in \Omega\), the function is actually the same as \[\begin{align} g \mapsto \min_{\substack{ x\in \mathbb{Z}^{n} \setminus \{ 0\} \\ \|x\| \le R_{\Omega}}} \|gx\|. \end{align}\]

Relation to SVP

Although the function is a continuous function of matrices \(g \in GL_n(\mathbb{R})\), finding the value computationally amounts to solving the SVP problem
